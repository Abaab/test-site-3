The Best Website Ever

Welcome! This website was created with my open-source GPL-licensed project, docs-to-pages .

tl;dr – this website was made:

  • For free
  • In a Google Drive folder
  • With an open-source deployment stack

The HTML for this website is fast, simple, and free of the terrible bloat that comes with services like Wix or SquareSpace. Plus, there's no banner here saying "made with docs-to-pages".

Of course, it's a little bit limited; the site can't be very responsive, and you have to set up all the hyperlinks yourself (links to docs will be converted into relative links).

…but if you want a free, simple way to make a website with minimal technical know-how, this ain't bad (I hope)!

Wahoo!
